    Plattformen

    Haben Sie Anmerkungen zu einer Portierung, oder wünschen Sie den Entwicklern Rückmeldungen zu geben, schicken Sie diese an die jeweilige Mailingliste.

    Unterstützte Platformen

    Die folgenden Plattformen werden derzeit von FreeBSD unterstützt:

    Plattform Mailingliste Support Tier
    amd64 (x86-64) freebsd-amd64@FreeBSD.org Tier 1
    ARM64 freebsd-arm@FreeBSD.org Tier 2
    ARM freebsd-arm@FreeBSD.org Tier 2
    i386 (IA-32) freebsd-current@FreeBSD.org Tier 1
    MIPS freebsd-mips@FreeBSD.org Tier 3
    PowerPC (ppc) freebsd-ppc@FreeBSD.org Tier 2
    sparc64 freebsd-sparc@FreeBSD.org Tier 2

    Nicht (mehr) unterstützte Plattformen

    Projektseite Mailingliste Support Tier
    FreeBSD/alpha Project freebsd-alpha@FreeBSD.org Tier 4 (Unterstützung eingestellt mit 7.0R)
    FreeBSD/ia64 Project freebsd-ia64@FreeBSD.org Tier 2 bis FreeBSD 10. Seither nicht mehr unterstützt.
    FreeBSD/RISC-V Project freebsd-riscv@FreeBSD.org Tier 3, beginnend mit FreeBSD 12
    FreeBSD/pc98 Project re-pc98@FreeBSD.org Tier 2 bis FreeBSD 11. Danach nicht mehr unterstützt.

    Für allgemeine Architekturfragen senden Sie bitte eine E-Mail an freebsd-arch@FreeBSD.org.
