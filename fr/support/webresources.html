Le système X Window

  • Les projets X.Org et XFree86™ fournissent à tous les utilisateurs de systèmes UNIX® pour architecture Intel, y compris FreeBSD, un excellent système X Window.
  • Le projet WINE a pour but de faire tourner des logiciels écrits pour Microsoft Windows® sur des OS UNIX pour processeurs Intel comme FreeBSD, NetBSD ou Linux.

Matériel

  • La FAQ de comp.answers pc-hardware-faq est une très bonne référence si vous voulez monter vous-même votre (vos) machine(s).
  • Intel Secrets -- What Intel Doesn't Want You To Know - beaucoup d'informations sur les processeurs Intel.
  • Aad Offerman's Chip List - matériel de référence sur les processeurs utilisés dans les clones PC.

Autres projets de systèmes d'exploitation

  • NetBSD est un autre UNIX libre basé sur 4.4BSD Lite et qui fonctionne sur différentes plates-formes.
  • OpenBSD est un autre OS dérivé de 4.4BSD qui met l'accent sur la sécurité.
  • Linux est un autre UNIX® libre.
  • Darwin est le système libre qui forme le coeur du système Mac OS X d'Apple.
  • Lites est un serveur basé sur 4.4BSD Lite et une librairie d'émulation qui fournit les fonctionnalités d'UNIX® à un système basé sur Mach.
  • Le projet GNU Hurd est un autre projet pour développer un OS libre proche d'UNIX®.
  • DragonFly BSD est basé sur FreeBSD 4.X mais a d'autres objectifs de développements que FreeBSD 5.X.
  • OpenSolaris est une variation open source du système d'exploitation Solaris de Sun Microsystems, qui est aussi un dérivé de BSD et fut longtemps appelé SunOS. La majeure partie du code source de base a été ouvert, mais le système d'exploitation complet est toujours également distribué selon l'ancienne méthode.
