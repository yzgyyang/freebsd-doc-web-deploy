    Platforms

    If you have comments about a port, or wish to provide feedback to the developers, send it to the relevant mailing list.

    Supported Platforms

    Here is a list of platforms that FreeBSD currently supports.

    Platform Mailing List Support Tier
    amd64 (x86-64) freebsd-amd64@FreeBSD.org Tier 1
    ARM64 freebsd-arm@FreeBSD.org Tier 2
    ARM freebsd-arm@FreeBSD.org Tier 2
    i386 (IA-32) freebsd-current@FreeBSD.org Tier 1
    MIPS freebsd-mips@FreeBSD.org Tier 3
    PowerPC (ppc) freebsd-ppc@FreeBSD.org Tier 2
    sparc64 freebsd-sparc@FreeBSD.org Tier 2

    Unsupported Platforms

    Project Page Mailing List Support Tier
    FreeBSD/alpha Project freebsd-alpha@FreeBSD.org Tier 4 (support discontinued as of 7.0R)
    FreeBSD/ia64 Project freebsd-ia64@FreeBSD.org Tier 2 through FreeBSD 10. Unsupported after.
    FreeBSD/RISC-V Project freebsd-riscv@FreeBSD.org Tier 3 beginning in FreeBSD 12
    FreeBSD/pc98 Project re-pc98@FreeBSD.org Tier 2 until FreeBSD 11. Unsupported after.

    For general architecture questions, mail freebsd-arch@FreeBSD.org
