      获取 FreeBSD

      选择一种架构

      大多数 FreeBSD 用户使用 amd64， i386 或 armv6 架构的硬件。

      现代个人电脑使用 amd64 架构， 包括那些使用 Intel® 品牌处理器的电脑。 超过 3 GB 内存的电脑应该使用 amd64。 如果电脑是更旧的32位型号， 使用 i386。 嵌入式设备和单片机， 例如 Raspberry Pi， Beagle Bone Black， Panda Board 和 Zen Board， 使用支持 ARMv6 和 ARMv7 处理器的 armv6 SD 卡映像。

      其他用户请参阅 FreeBSD 支持的平台的 完整列表 。

      选择一个映像

      安装 FreeBSD 有许多种选择， 包括从 CD (disc1)、 DVD (dvd1)、 网络安装映像 (仅引导) 和 USB 映像中启动 FreeBSD 安装程序。 对于最近的 FreeBSD 版本， 虚拟机映像， 以及用于嵌入式平台的 SD 卡映像也可供下载。

      FreeBSD 部署数据

      尽管 FreeBSD 并不收集部署数据， 获取统计信息是十分重要的。 请考虑安装 sysutils/bsdstats 包， 这将收集硬件和软件的统计数据， 以帮助开发者理解如何更好地进行开发。 收集的信息可在 bsdstats.org 网站查阅。

      安装 FreeBSD

      安装 FreeBSD 有许多种选择， 包括从 CD-ROM、 DVD、 USB 磁盘、 匿名 FTP、 HTTP 以及 NFS 安装。 请在下载完整的 FreeBSD 发行版之前通读 FreeBSD安装说明 。

      购买 FreeBSD

      FreeBSD 可以在 FreeBSD Mall 或其它的 CD-ROM 和 DVD 发行商 处取得 CD-ROM 或 DVD。

      过去的发行版

      若要下载过去的发行版， 请到 FTP 存档 下载。

      FreeBSD 衍生的操作系统发行版

      FreeBSD 被广泛用于其它商业和开源操作系统的组成部分。 以下的项目被广泛使用并尤其使 FreeBSD 用户感兴趣。

      • FreeNAS 是一个基于 FreeBSD 的开源存储平台， 它支持在 Windows、 Apple 和类 UNIX 系统之间分享文件。

      • TrueOS （前身为 PC-BSD） 是一个具有图形化安装程序的 FreeBSD 衍生操作系统， 其桌面选项安装 Lumina 桌面环境， 一个完全采用 BSD 协议的桌面环境。 它令人印象深刻的桌面工具旨在方便一般电脑用户使用。

      • pfSense 是一个基于 FreeBSD 的定制发行版， 为防火墙和路由器使用专门定制。
