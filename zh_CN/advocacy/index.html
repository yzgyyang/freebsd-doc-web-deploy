    这个页面将提供给您这方面更多的信息。

    使用 FreeBSD 的网站

    • Daemon News 是一个关于 BSD 操作系统的电子出版物。 它的目标是帮助 FreeBSD、 OpenBSD， 以及 NetBSD 社区的用户。
    • BSD Freak 是一个从用户视角提供关于 BSD 操作系统教程、 文章， 以及定期刊物的新网站。
    • 由 Netcraft 提供的 托管商表现 记录了主要的 web 托管服务的可靠性， 这些托管商中有很多使用 FreeBSD。
    • The Open Directory Project 的目标是， 通过为数众多的志愿者编辑， 来建立尽可能详尽的 web 目录。
    • FreeBSD、 Linux 以及 Windows 2000 的对比 是一篇比较三种操作系统可靠性、 性能、 支持情况、 整体拥有成本等方面的文章。 (其内容已经有些过时)
    • BSDCan， 在加拿大渥太华举办的年度 BSD 会议。
    • EuroBSDCon， 在欧洲举办的年度 BSD 会议。
